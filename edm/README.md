# Manila EDM Directory site

Static public renderer for the `manila-directory/edm` data layer.

- `index.html` — simple public entry point
- `directory.html` — browse/search surface
- `entry.html` — full artist card route
- `assets/directory-data.json` — generated payload; do not hand-edit

Local preview:

```bash
python3 -m http.server 8777
```
